What is aGAN?

aGAN is a machine learning project that I started working on in Fall 2020 as part of the CVU Nexus Program. It's still a work-in-progress, but is intended to be a machine learning algorithm for visualizing data from complex physical systems (wave propagation, gas diffusion, fluid flow, etc.) with abstract art.
